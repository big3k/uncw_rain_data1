Experiments: Linear regression with pdf-corrected sounders, for all sounders (noaa18, noaa19, metopa, metopb, & npp)

1. Experiment Design

Same as Model 1, but with experiments with pdf-matching approach to correct all sounders (n18, n19, metopa, metopb, and npp)before modeling. Experiments were also performed with the imager data corrected to gmi.


2. Data Preparation

All the data rows are randomly shuffled to evenlly distribute the time_diffs for the samples. Then 60% of the data are used for training, and 40% for validation.

3. Results


Experiments. Sounder pdf-matched to GMI, linear model, no intercept

  • 1) All sounders combined, all system sizes
  • 2) All sounders combined, system sizes < 500
  • 3) All sounders combined, system sizes >= 500
  • 4) Noaa18 only, system sizes < 500
  • 5) Noaa18 only, system sizes >= 500
  • 6) Noaa19 only, system sizes < 500
  • 7) Noaa19 only, system sizes >= 500
  • 8) Metopa only, system sizes < 500
  • 9) Metopa only, system sizes >= 500
  • 10) Metopb only, system sizes < 500
  • 11) Metopb only, system sizes >= 500
  • 12) NPP only, system sizes < 500
  • 13) NPP only, system sizes >= 500
