Experiments: Linear regression with pdf-corrected sounders, for all sounders (noaa18, noaa19, metopa, metopb, & npp)

1. Experiment Design

Same as Model 7, i.e., with pdf-matching approach to correct all sounders (n18, n19, metopa, metopb, and npp) and the propagated imagers to GMI before modeling. The difference now is that another GMI PDF-matching is performed on the predicted values.


2. Data Preparation

All the data rows are randomly shuffled to evenlly distribute the time_diffs for the samples. Then 60% of the data are used for training, and 40% for validation.

3. Results


Experiments. Sounder pdf-matched to GMI, linear model, no intercept

  • 1) All sounders combined, all system sizes
  • 2) All sounders combined, system sizes < 500
  • 3) All sounders combined, system sizes >= 500
  • 4) All sounders but NPP, system sizes < 500
  • 5) All sounders but NPP, system sizes >= 500
  • 6) NPP only, system sizes < 500
  • 7) NPP only, system sizes >= 500
